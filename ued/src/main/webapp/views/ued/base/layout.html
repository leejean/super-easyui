一、特性

  1. 提炼和服务淘实惠后台产品的交互语言和视觉风格。
  2. jquery、easyui、easyloader上封装高质量UI库
  3. 升级版本不影响用户使用（用户端缓存自动清除方式）
  4. 支持IE8+

二、文档支持

  1. 开发规范及开发指南文档（给前端人员使用sass、webpack等）
  2. 前端框架的基本结构（demo）
  3. 组件封装和使用指南（供后端人员使用）
  4. 目录结构规范说明
  5. 组件库方式提供方式（API）
  6. 安装使用
  7. 升级指南
  8. 更新日志

三、目录结构

目录结构
  • webapp
    • static
      • tsh(淘实惠PC前端框架)
      • xxx项目（项目名）
        • js（项目js）
          • common（项目公用js）
          • goods(index.js;list.js;add.js;edit.js;info.js)
          • css（项目css）
          • images（项目图片）
    • views
      • home.html
      • common(项目公共页面)
      • error
        • 404.html
        • 500.html
      • goods
        • index.html（首页）
        • list.html（列表页）
        • edit.html(编辑页)
        • add.html(添加页)
