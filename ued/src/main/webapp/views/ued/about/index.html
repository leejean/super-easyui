关于

一、特性

  1. 提炼和服务淘实惠后台产品的交互语言和视觉风格。
  2. 基于jquery1.11.3、easyui1.4.5上封装高质量UI库
  3. 升级版本不影响用户使用（用户端缓存自动清除方式）
  4. 浏览器支持：IE8+

二、文档支持

  1. 开发规范及开发指南文档（给前端人员使用sass、webpack等）
  2. 前端框架的基本结构（demo）
  3. 组件封装和使用指南（供后端人员使用）
  4. 目录结构规范说明
  5. 组件库方式提供方式（API）
  6. 安装使用
  7. 升级指南
  8. 更新日志

三、目录结构

目录结构
  • webapp
    • static
      • tsh(淘实惠PC前端框架)
      • xxx项目（项目名）
        • js（项目js）
          • common（项目公用js）
          • goods(index.js;list.js;add.js;edit.js;info.js)
          • css（项目css）
          • images（项目图片）
    • views
      • home.html
      • common(项目公共页面)
      • error
        • 404.html
        • 500.html
      • goods
        • index.html（首页）
        • list.html（列表页）
        • edit.html(编辑页)
        • add.html(添加页)

三：使用示例

使用项目时，仅需要引入static内tsh目录。此目录内容为框架，会不断更新。以满足各种业务需求。 

  <!DOCTYPE html>
  <html>
  <head>
      <meta charset="UTF-8">
      <meta http-equiv="X-UA-Compatible" content="IE=Edge,chrome=1" />
      <meta name="renderer" content="webkit">
      <title>淘实惠管理系统</title>
      <link rel="stylesheet" type="text/css" href="../../static/tsh/1.0.0/themes/easyui.css?gversion=1.0.0">
      <link rel="stylesheet" type="text/css" href="../../static/tsh/1.0.0/themes/style.css?gversion=1.0.0">
      <script src="../../static/tsh/1.0.0/jquery.min.js?gversion=1.0.0"></script>
      <script src="../../static/tsh/1.0.0/jquery.easyui.min.js?gversion=1.0.0"></script>
      <script src="../../static/tsh/1.0.0/jquery.easyui.extend.min.js?gversion=1.0.0"></script>
  </head>
  <body style="margin:0 20px;">
  <div id="tsh-title">
    <h3><span class="t1">标题</span><span class="t2">副标题</span>  </h3>
    <div class="title-border"></div>
  </div>
  </body>
</html>
