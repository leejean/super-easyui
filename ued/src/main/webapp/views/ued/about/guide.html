使用指南

一、快如上手

各个项目组每次只需更新替换static/tsh整个文件夹。此文件夹为淘实惠前端框架。其余文件均为示例代码。仅供参考

一、tsh框架目录结构

目录结构
  • webapp
    • static
      • tsh(淘实惠PC前端框架)
      • xxx项目（项目名）
        • js（项目js）
          • common（项目公用js）
          • goods(index.js;list.js;add.js;edit.js;info.js)
          • css（项目css）
          • images（项目图片）
    • views
      • home.html
      • common(项目公共页面)
      • error
        • 404.html
        • 500.html
      • goods
        • index.html（首页）
        • list.html（列表页）
        • edit.html(编辑页)
        • add.html(添加页)
